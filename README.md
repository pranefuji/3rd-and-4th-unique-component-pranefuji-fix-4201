# 3rd-and-4th-Unique-Component-Pranefuji-Fix-4201
This mod is a temporary update that I made for a mod last updated 2025-07-28 by EmperorPinguin, to work with unciv version 4.20.1
I am sharing this version in the meantime for users to enjoy EmperorPinguin great work between updates.

Changes have been made for the following
The (building) Json for Hippodrome was depreciated as of 4.15.11
The (wonder) Temple of King Solomon was depcreciated as of 4.18.16
The (promotion) Dao Swordsman was depreciated as of 1.16.14
The (improvement) Arepa House, contained a parameter "All", which does not fit parameter type terrain feature

The original mod adds a 3rd and 4th unique component to each civilization in the G&amp;K base game.
It also adds extra components to various extension mods.
